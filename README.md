# Calculator App

The app is an functional calculator, based on the iOS Calculator, makes: sum, subtraction, multiply, divide and clear the count.

## Requirements

```
iOS 13+
Xcode 11
Swift 5
UIKit
```

## This class uses:

```
UIStackView
```
